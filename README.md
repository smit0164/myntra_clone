# Myntra Clone

A responsive e-commerce website clone built with HTML and CSS, replicating the design and layout of the popular fashion platform Myntra.

## 📱 Project Overview

This project is a front-end recreation of the Myntra online shopping platform. It features a clean, modern interface with a professional navigation system, product display sections, and comprehensive footer with links to important pages.

## ✨ Features

- **Responsive Header Navigation** - Stylish navbar with multiple category links (Men, Women, Kids, Home, Beauty, GenZ, Studio)
- **Search Functionality** - Integrated search bar with icon in the header
- **User Actions** - Quick access buttons for Profile, Wishlist, and Shopping Bag
- **Product Sections** - Two main product showcase areas:
  - NEW OFFERS - Featured promotional items
  - SHOP BY CATEGORY - Products organized by category
- **Professional Footer** - Multiple columns with:
  - Online Shopping links
  - Customer Policy information
  - Copyright information
- **Modern Design** - Clean, minimalist interface following current web design trends
- **Material Symbols** - Google Material Icons for enhanced UI elements

## 🚀 How to Run

1. **Clone the repository** (or download the files)
   ```bash
   git clone <repository-url>
   cd myntra_clone
   ```

2. **Open in Browser**
   - Simply open `index.html` in your web browser
   - No server or build tools required - it's pure HTML/CSS

3. **Alternative: Use Live Server**
   - If using VS Code, install the Live Server extension
   - Right-click on `index.html` and select "Open with Live Server"
   - This provides hot reload during development

## 🔧 CSS Classes Overview

### Header Components
- `.logo_container` - Myntra logo area
- `.navbar_container` - Navigation menu
- `.search_container` - Search bar
- `.action_bar` - User actions (Profile, Wishlist, Bag)

### Content Components
- `.banner_container` - Hero banner section
- `.container_heading` - Section headings (NEW OFFERS, SHOP BY CATEGORY)
- `.offer_container` - Flexible grid for product display
- `.sale_item` - Individual product items

### Footer Components
- `.footer_container` - Main footer content area
- `.footer_column` - Footer navigation columns
- `.footer_bottom` - Copyright and bottom footer section
